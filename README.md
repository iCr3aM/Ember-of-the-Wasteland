# Ember of the Wasteland

[![Ren'Py](https://img.shields.io/badge/Ren'Py-8.0+-blue.svg)](https://www.renpy.org/)
[![Status](https://img.shields.io/badge/Status-Beta-orange)]()
[![License](https://img.shields.io/badge/License-Proprietary-red)]()
[![Platform](https://img.shields.io/badge/Platform-Windows%20%7C%20Linux%20%7C%20macOS-lightgrey)]()

> A post-apocalyptic survival RPG inspired by *Neo Scavenger* and the *Fallout* series.  
> Manage resources, engage in turn‑based combat, explore dynamic events, and uncover the secrets of the wasteland.

**Made by Cr3aM** — built with the Ren'Py engine.
